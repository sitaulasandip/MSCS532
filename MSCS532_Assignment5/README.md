# Assignment 5: Quicksort Algorithm

This folder contains deterministic and randomized Quicksort implementations, unit tests, an empirical benchmarking program, sample benchmark results, and a detailed performance-analysis report.



## Requirements

- Python 3.10 or newer recommended
- No third-party packages are required

## Run the Quicksort Demonstration

```bash
python quicksort.py
```

